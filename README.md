# Flutter Project

This is a Blackminystudio's Project "TacklePay App", help to manage All your finances.

## Setup

1. Clone the repo:

   ```bash
   git clone https://github.com/blackminystudio/tacklepay.git
   ```

   ```bash
   cd tacklepay
   ```

2. Install dependencies:

   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```
